Cara Menjalankan Dashboard:
1. Pastikan Library yang Diperlukan Sudah Terinstal:
Anda memerlukan beberapa library Python berikut:

streamlit
pandas
plotly

Instal library tersebut dengan perintah berikut:
pip install streamlit pandas plotly

2. Siapkan File CSV:
Pastikan file all_data.csv berada di direktori yang sama dengan file dashboard.py. File CSV ini diperlukan karena dashboard membaca data dari file tersebut.

3. Jalankan Dashboard:
Gunakan perintah berikut di terminal untuk menjalankan dashboard:
streamlit run "dashboard.py"

4. Akses Dashboard:
Setelah perintah berhasil dijalankan, server lokal akan aktif, dan Anda dapat mengakses dashboard melalui browser di alamat berikut:
http://localhost:8501
